# Limboda

Water management and information system built with Next.js.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
npm start
```

## Project Structure

- `app/` - Next.js App Router directory
  - `layout.tsx` - Root layout with navigation
  - `page.tsx` - Home page
  - `water-readings/` - Water readings page
  - `news/` - News page
  - `bra-att-ha/` - Information resources page
  - `admin/` - Admin panel (placeholder)

## Pages

- **Home** - Landing page
- **Water Readings** - Management of water meter readings
- **News** - Latest updates and announcements
- **Bra att ha** - Useful information and resources
- **Admin** - Admin panel (role-gated access to be implemented)
